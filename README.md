# 🏕️ Camp Half-Blood

Sistema inspirado no universo de **Percy Jackson**, desenvolvido para gerenciar informações de um acampamento de semideuses.

## 📖 Sobre o projeto

O **Camp Half-Blood** é uma aplicação web desenvolvida como **projeto individual do 1º semestre do curso de Análise e Desenvolvimento de Sistemas da SPTech (São Paulo Tech School)**.

O objetivo do projeto foi praticar conceitos fundamentais de **desenvolvimento web**, como criação de interfaces, lógica de programação e integração entre frontend e backend utilizando JavaScript.

O sistema permite que usuários se registrem no acampamento, realizem quizzes e acompanhem informações personalizadas em seu painel.

## 🚀 Tecnologias utilizadas

* JavaScript
* Node.js
* HTML
* CSS
* MySQL
* API Web-Data-Viz (para visualização de dados)

## ⚙️ Funcionalidades

* Cadastro de usuários
* Login de usuários
* Quiz de **conhecimentos sobre mitologia grega**
* Quiz de **personalidade**
* Dashboard individual do usuário
* Exibição do **desempenho no quiz de conhecimentos**
* Exibição das **características da personalidade do usuário**
* **Card aleatório do dia** com uma história sobre um deus grego
* Alteração de **foto de perfil do usuário**

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar **desenvolvimento web full stack**, utilizando **JavaScript no frontend, Node.js no backend e MySQL para persistência de dados**, além da utilização de ferramentas de **visualização de dados para o dashboard**.
